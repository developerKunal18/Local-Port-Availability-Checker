# Local Port Availability Checker

A Python script that checks whether a local port is available.

## Description
This project helps developers verify if a port is free before running local servers or applications.

## Features
- Fast port checking
- Useful for backend development
- Beginner-friendly networking tool

## Technologies Used
- Python
- socket module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python port_checker.py

4. Enter port number

## Example

Input:
5000

Output:
Port 5000 is available
